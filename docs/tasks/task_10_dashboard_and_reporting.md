# Task 10: Dashboard and Reporting

## Deskripsi
Fitur analitik dan dashboard untuk melihat ringkasan operasional rental.

## Kebutuhan
1. **Agregasi Data:** Menampilkan jumlah rental berjalan, pendapatan, kendaraan aktif, dan notifikasi (perawatan/asuransi).
2. **Data Dummy (Seeder):** Memastikan semua seeder dari task 1-9 terhubung dengan baik agar dashboard menampilkan data dan grafik yang komprehensif.
