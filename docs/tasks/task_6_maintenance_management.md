# Task 6: Maintenance Management

## Deskripsi
Fitur penjadwalan dan pencatatan riwayat perawatan kendaraan.

## Kebutuhan
1. **Riwayat Perawatan:** CRUD tabel maintenance_records.
2. **Penjadwalan:** Mencatat tanggal maintenance dan next_maintenance_date.
3. **Data Dummy (Seeder):** Buat data dummy riwayat perawatan untuk beberapa kendaraan untuk melihat visualisasi jadwal servis.
